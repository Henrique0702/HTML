protocolo

Na internet tanto hardware como o software pode ser diferentes.

Apesar das diferenças para a troca de informações devemos usar o mesmo protocolo.

Podemos definir protocolos como regras para troca de informações.

IP Internet Protocol
Protocolo de internet
responsáveis pelo endereçamento, pelo destino dos blocos de dados
TCP Transfer Control Protocol
Protocolo de controle de tranferência
Responsável pelo envio dos blocos de dados e integridade do conteúdo

Temos outros protocolos como

  1. FTP File Transfer Protocol resposável pelo download de arquivos
  2. HTTP Hiprtext Transfer Protocol
    • Protocolo de tranferência de hipertexto
    • Responsável pela navegação através de links
    • simples que fez sucesso gerando a WWW
  3. SMTP simple Mail Transfer Protocol Protocolo para email básico
